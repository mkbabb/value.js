The new demo tree, current `style.css`/feature styles, docs Markdown imports, `plugins/vite-source-export.ts`, App/router/color pipeline boundaries, the eleven feature-local route leaves including first-class `/about` and `/easing`, dead barrels, and large components identified by W2.
