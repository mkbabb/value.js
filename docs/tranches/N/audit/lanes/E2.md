## 5. Typecheck status (3 RED errors)

Confirmed by running `node scripts/check-types.mjs`:

```
demo/@/components/custom/color-picker/controls/ComponentSliders.vue(117,50): error TS2307:
  Cannot find module '@mkbabb/glass-ui/glass-carousel' or its corresponding type declarations.

demo/@/components/custom/mix/MixSourceSelector.vue(4,10): error TS2305:
  Module '"@mkbabb/glass-ui/tabs"' has no exported member 'BouncyTabs'.

demo/@/components/custom/panes/PaneSegmentedControl.vue(18,10): error TS2305:
  Module '"@mkbabb/glass-ui/tabs"' has no exported member 'BouncyTabs'.
```

### Root causes

**Error 1 — wrong subpath:** `ComponentSliders.vue:117` imports `GlassCarousel, GlassCarouselItem` from `@mkbabb/glass-ui/glass-carousel`. The correct subpath in glass-ui 3.10.1's `package.json` is `./carousel` (not `./glass-carousel`). Furthermore, `dist/carousel.d.ts` exports only `Carousel`, `CarouselContent`, `CarouselDots`, `CarouselItem`, `CarouselNext`, `CarouselPager`, `CarouselPrevious`, `GlassCarouselPager` — no `GlassCarousel` or `GlassCarouselItem` named exports exist.

**Errors 2 & 3 — renamed export:** `BouncyTabs` was renamed to `SegmentedTabs` in glass-ui during tranche AX.W53. The tabs index now only exports `SegmentedTabs` and its associated types. The correct replacement is `import { SegmentedTabs } from "@mkbabb/glass-ui/tabs"`.

### Fix surface

| File | Line | Wrong | Correct |
|------|------|-------|---------|
| `demo/@/components/custom/color-picker/controls/ComponentSliders.vue` | 117 | `@mkbabb/glass-ui/glass-carousel` → `{GlassCarousel, GlassCarouselItem}` | `@mkbabb/glass-ui/carousel` → `{Carousel, CarouselItem, ...}` (verify names) |
| `demo/@/components/custom/mix/MixSourceSelector.vue` | 4 | `BouncyTabs` from `@mkbabb/glass-ui/tabs` | `SegmentedTabs` from `@mkbabb/glass-ui/tabs` |
| `demo/@/components/custom/panes/PaneSegmentedControl.vue` | 18 | `BouncyTabs` from `@mkbabb/glass-ui/tabs` | `SegmentedTabs` from `@mkbabb/glass-ui/tabs` |
