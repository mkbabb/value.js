## 3. `npm run typecheck` — FAIL (3 errors)

Command: `node scripts/check-types.mjs` (runs vue-tsc across two split tsconfig programs)

**Result: exit 1 — 3 value.js-owned type errors in the demo graph.**

### Error 1 — `glass-carousel` subpath does not exist

```
demo/@/components/custom/color-picker/controls/ComponentSliders.vue(117,50):
  error TS2307: Cannot find module '@mkbabb/glass-ui/glass-carousel' or its
  corresponding type declarations.
```

File: `demo/@/components/custom/color-picker/controls/ComponentSliders.vue:117`
```ts
import { GlassCarousel, GlassCarouselItem } from "@mkbabb/glass-ui/glass-carousel";
```

Root cause: `@mkbabb/glass-ui` (local `file:../glass-ui`) exports `./carousel` (not
`./glass-carousel`). The carousel barrel lives at `src/carousel.ts` and is mapped as
`"./carousel"` in `glass-ui/package.json`. The `glass-carousel` subpath does not appear
anywhere in the exports map.

The correct import is `@mkbabb/glass-ui/carousel`.  
The correct exports are: `Carousel`, `CarouselContent`, `CarouselItem`, `CarouselNext`,
`CarouselPrevious`, `CarouselDots`, `CarouselPager`, `GlassCarouselPager`, `useCarousel`.
`GlassCarousel` and `GlassCarouselItem` are not exported by `./carousel` either — the
nearest candidates are `Carousel` and `CarouselItem`.

### Errors 2 & 3 — `BouncyTabs` renamed to `SegmentedTabs`

```
demo/@/components/custom/mix/MixSourceSelector.vue(4,10):
  error TS2305: Module '"@mkbabb/glass-ui/tabs"' has no exported member 'BouncyTabs'.

demo/@/components/custom/panes/PaneSegmentedControl.vue(18,10):
  error TS2305: Module '"@mkbabb/glass-ui/tabs"' has no exported member 'BouncyTabs'.
```

Files:
- `demo/@/components/custom/mix/MixSourceSelector.vue:4`
- `demo/@/components/custom/panes/PaneSegmentedControl.vue:18`

Both contain:
```ts
import { BouncyTabs } from "@mkbabb/glass-ui/tabs";
```

Root cause: The `@mkbabb/glass-ui/tabs` subpath exports only `SegmentedTabs` (and its
associated types). `BouncyTabs` was retired in glass-ui tranche AX.W53 — the component was
renamed and the entire `BouncyToggle`/`BouncyTabs`/`UnderlineTabs` trio collapsed onto
`SegmentedTabs` with a `variant` axis.

Confirmed from `glass-ui/src/components/custom/tabs/index.ts`:
```ts
export { default as SegmentedTabs } from "./SegmentedTabs.vue";
export type { SegmentedTabOption, SegmentedTabsVariant, SegmentedTabsProps, SegmentedTabsResponsive } from "./SegmentedTabs.vue";
```

And from `glass-ui/dist/components/custom/tabs/index.d.ts` (the resolved dist types):
```ts
export { default as SegmentedTabs } from "./SegmentedTabs.vue";
```

No `BouncyTabs` export exists anywhere in the glass-ui tree.
