## Scope

W7 is the close ceremony. It opens only after W6 has landed or re-scoped, so no audit lane runs over an open cross-repo dependency (`audit/HARDEN-4`, `audit/HARDEN-6`).

### Read-only close audit lanes

Per the precept `SPEC §Close` close-ceremony pattern, adapted to value.js as a first tranche. The lanes are read-only and are the documented exception to the six-agent implementation ceiling.

1. **plan-vs-actual** — every wave's planned scope against what landed; `PROGRESS.md` matches reality.
2. **substrate-without-consumer** — any helper, token, `@apply` class, or component A introduced without a consumer; flag visual-load-bearing-ness, not only consumer count.
3. **doc-drift** — `A.md`, `findings.md`, `coordination/Q.md`, the wave specs reconciled against the shipped tree.
4. **idiomatic-gestalt** — the demo's glass-ui consumption is idiomatic; no fossil rebuilt-by-hand surface survived; invariants A1–A4 hold.
5. **performance** — bundle size, the goo-blob + aurora frame budget (`audit/HARDEN-5` raised the dual-WebGL contention concern); a frame-budget artefact.
6. **visual-runtime** — Playwright re-probe, binding (A ships visual changes): ≥3 viewports light + dark, animation-frame samples on every state transition A introduced or modified, contrast measurement for every interactive surface A touched, and a per-story consumption sweep confirming every CVA/token/utility A migrated is consumed canonically.
7. **integrity sweep** — `git reflog` since A open for unauthorized agent-attributed mutating git operations; `git stash list` clean across the primary tree and every worktree A used; `git log` on `docs/precepts` for unexpected submodule changes.

### Close ceremony

1. `FINAL.md` — close report citing every wave's commits, the gate evidence, the Playwright artefacts, and the disposition of every `research/Aa..Ae` finding (landed / retired-with-rationale / routed to `coordination/Q.md`).
2. Reconcile `PROGRESS.md` against reality; run the close-honesty checklist (every `FINAL.md` claim grounded in `PROGRESS.md` or a cited artefact; every gate marked MET has a resolving evidence path; every cross-tranche debt entry names a destination).
3. Pin the `docs/precepts` SHA A acknowledged in the close record, so a later precepts advance does not retroactively desync A's close (`coordination/Q.md §6`).
4. Confirm the `coordination/Q.md §0` contested boundary is resolved — glass-ui Q deleted Q.W1 Lane C and Q.W2 Lane B — or record it as an open cross-repo item with a named owner.
