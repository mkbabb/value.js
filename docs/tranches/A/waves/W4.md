3. `EditDrawer` save/cancel — wrap the bare `<Check>`/`<Undo2>` SVGs in `<button type="button">` (or glass-ui `Button variant="ghost" size="icon"`); the `focus-visible` classes on a raw SVG never fire.

4. ### Lane C — `Dock.vue` + `App.vue` decomposition (Ae-1, Ae-2)

1. `Dock.vue` (426 lines) — extract `useDockLayers()`, `useDockAdminMode()`, `DockViewSelect.vue`, `DockMainLayer.vue`. Target `Dock.vue` ≤ ~120 lines.
2. `audit/HARDEN-4` decomposition hazards, binding on this lane:
   - `usePopupMutex` is a single shared mutex. If `DockViewSelect.vue` calls `usePopupMutex` a second time, the single-open contract breaks silently. `DockViewSelect.vue` must receive its open state as a `v-model:open` prop from `Dock.vue`, not create its own mutex model.
   - Dock composables touching `dockRef` must receive it explicitly, not re-query it.
   - The `immediate:true` layer-dispatch watch needs its three dependencies passed as reactive refs with pinned call order.
   - W4 must not change the `"auroraConfig"` provide contract — W6 owns it.
3. `App.vue` (393 lines) — extract `usePaletteManagerWiring()`, `useAtmosphere()`, and move the color-dedup block to a color composable. Target `App.vue` script ≤ ~150 lines.

### Lane D — `<PaneSlot>` flatten + `ConfigSliderPane` (Ae-3, Ae-4, Ae-5, Ae-6)

1. Introduce `<PaneSlot :side="...">` wrapping `Transition` + `KeepAlive` + `component :is`; collapse the triple-nested pane dispatch in `App.vue` and delete the desktop `v-if` ladder.
2. Remove the inert single-child wrapper divs; convert the `App.vue:90-93` inline 5-property `:style` hack to a class.
3. Merge `AuroraPane.vue` + `BlobPane.vue` into one local `ConfigSliderPane` parameterized by `{ config, sections, defaults, extraControls? }`. This is a demo-local component for W4; W6 migrates it onto glass-ui's existing `./configurator` surface (`ConfiguratorRow` + `useConfiguratorState`), which `audit/HARDEN-4` confirmed glass-ui already ships — the gap is non-adoption, not a missing primitive.
4. Delete `PaneSegmentedControl.vue`'s duplication against the inline `Dock.vue` `BouncyTabs` block.
