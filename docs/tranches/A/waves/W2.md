### Lane A ‚Äî dock calc-chain de-tangle + z-tier adoption (Ab-14, Ab-15)

1. Break the fold-back in the `style.css:50-76` dock layout chain. `--dock-pos` (`:59`) back-derives the dock position from `--content-max-h`, while the desktop and 21:9 media queries (`:67`, `:73`) replace `--content-max-h` with independent `clamp()` formulas ‚Äî editing the desktop clamp silently moves the dock. Recompute `--dock-pos` purely from the dock's own footprint (`--dock-inset` + `--dock-h` + `--dock-gap`); `--content-max-h` becomes a leaf nothing else consumes.
2. Replace the magic literals inside the chain (`0.75rem` at `:52`, the `+0.5rem` at `:99`) with named tokens. The chain uses W2-owned layout tokens, not W3 design tokens ‚Äî `audit/HARDEN-3` confirmed this ordering is clean and W2/W3 do not fight here.
3. Finish the grid migration the `style.css:91` comment claims is done ‚Äî `.app-layout` should derive row sizing from `grid-template-rows`, not a `calc()` padding.
4. Replace bare `z-index: 1/2` and `z-[1]` in `PaneHeader.vue`, `AuroraPane.vue`, `BlobPane.vue`, `GooBlob.vue` with glass-ui z-tier tokens.

### Lane B ‚Äî unscoped-style split + `style.css` extraction (Ab-8, Ab-9, Ab-10)

1. Split the six unscoped `<style>` blocks (`WatercolorDot`, `Markdown`, `EditDrawer`, `PaletteDialog`, `PaletteDialogHeader`, `CurrentPaletteEditor`) ‚Äî keep one unscoped block for the Vue `*-enter/*-leave` transition classes and `:has()` portal selectors, move every plain component selector into a scoped block.
2. Move the component-specific recipes out of `style.css` ‚Äî `.section-subtitle`, the `.filter-section`/`.filter-option` block ‚Äî into the components that render them. `style.css` keeps `@theme`, the layout-token `:root`, `.app-layout`, `.pane-container`, the `@layer base` reset.
3. The `underline-tabs` reka-ui attribute override (`style.css:145-165`) routes to a glass-ui Tabs `underline` variant (`coordination/Q.md ¬ß3`); W2 removes the demo-side global override once the variant is consumable, otherwise records a marker.

### Lane C ‚Äî deprecated-CSS retire (Ab-11, Ab-12, Ab-13)

1. `style.css:204-208` ‚Äî replace the `-webkit-box` line-clamp triad with the standard `line-clamp`.
2. Delete the dead `-webkit-overflow-scrolling: touch` (`PointerDebugOverlay.vue:318`); replace `::-webkit-scrollbar { display:none }` (`PaletteControlsBar.vue:161`) with glass-ui `.scrollbar-hidden`.
3. `ExtractControls.vue:152-225` ‚Äî replace the two native `<input type=range>` sliders (8 vendor-pseudo rules) with the reka-ui Slider used elsewhere.
